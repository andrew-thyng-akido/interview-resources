# ToDoIt

## About
A simple todo list application built with React, TypeScript, and Tailwind CSS. It allows users to add, delete, and toggle the completion status of their tasks. The app is designed to be responsive and visually appealing, making it easy for users to manage their daily tasks.

## Installation
`npm install`

## Running the app
`npm run dev`

## Stack
- React
- TypeScript
- Tailwind CSS
- Vite
